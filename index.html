Modules

common Common contains utilities that is frequently used by the emulator.
cpu CPU provides access to read/write operations on the emulated CPU's general purpose registers (GPRs).
events Events module provides abilities to register callbacks at certain events in the emulator.
ipc.context IPC context provides utilities for manipulating an IPC request.
kernel Kernel module provides access to the emulated system's kernel object (read/write/...)
mem Mem module provides read/write operations on current memory address space.
std Std contains standard functions related to Symbian development's data structure.
