◆ commit()

bool eka2l1::kernel::chunk::commit ( uint32_t  offset,
size_t  size 
)

Commit to a disconnected chunk.

Offset and size SHOULD be aligned with the page size, else this will results unwanted behavior. E.g commit(page_size - 1, 2), should commit both the first and second page, since the offset is at the first page, and the commit contains both last and first byte of two page. This will results two WHOLE pages being allocated.

Parameters
offsetThe offset to commit to.
sizeThe size to commit.
Returns
false if failed to commit.
