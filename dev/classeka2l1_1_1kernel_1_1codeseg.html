Create a new codeseg.

code_load_addr and data_load_addr will be replace with load address.

Ideally, apps can use this to create a code portion that will resides in RAM code address. Don't pass relocation table, export table, and import table, no data size and only code base with random numbers. UID should be unique too, which you can choose any numbers that is far from 0x20000000 (UID0). After that, call CodesegCreate and the code_addr should be return for you.
