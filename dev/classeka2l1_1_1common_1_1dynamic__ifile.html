Detailed Description

A wrapper around ifstream, support reading many encodings.

This is a wrapper of fstream, supports reading UTF-8 and UCS-2 file to either string or an 16-bit string

It was born to deal with system files in Symbian, where most of them are saved as an UCS2 file.
