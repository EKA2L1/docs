void eka2l1::epoc::event_fifo::do_purge ( )
protected

Purge some events on the queue.

When the queue reached the maximum declared, it will purge some events (if the condition is met) to free space for other events to be queued.

On hardware, this is to prevent memory being wasted due to no users requests a notification for a long time. We should do this too, there is no reason not to do it.
