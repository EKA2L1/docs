◆ add_or_get()

drivers::handle eka2l1::epoc::bitmap_cache::add_or_get ( epoc::bitwise_bitmap *  bmp)

Add a bitmap to texture cache if not available in the cache, and get the driver's texture handle.

If the cache is full, this will find the least used bitmap (by sorting out last used timestamp). Also, since bitwise bitmap modify itself by user's will without a method to notify the user, this also hashes bitmap data (using xxHash), and will reupload the bitmap to driver if the texture data is different.

Parameters
bmpThe pointer to bitwise bitmap.
Returns
Handle to driver's texture associated with this bitmap.
