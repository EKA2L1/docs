uint16_t  crr_blck_idx 
)

Get the data in the index of a buffer block in the SIS.

The function assumes that data has small size, and will load it into a buffer. Of course, if the buffer has size of something like 200 MB, it will crash.

This function is usually used for extracting text file.

Parameters
data_idxThe index of the source buffer in block buffer.
crr_block_idxThe block index.
Returns
A vector contains binary data, uncompressed if neccessary.
