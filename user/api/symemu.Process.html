readProcessMemory()¶

Read at the specified address in the process’s address space.

The function can throw an exception in case the memory at the given address in the memory space is not yet mapped, or the size of the data exceeds the mapped memory size from that address fowards.

Parameters
addr: int

The address that we want to read from.

size: int

Size of data we want to read.

Returns
str

A string contains the data.

writeProcessMemory()¶

Write at the specified address in the process’s address space.
